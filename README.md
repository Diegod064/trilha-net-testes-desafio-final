# Blindando Seu Código com TDD e Testes Unitários Usando .NET Core

Este projeto é parte do módulo "Blindando Seu Código com TDD e Testes Unitários Usando .NET Core", onde exploramos como os testes unitários podem proteger e fortalecer nosso código, além de aprender sobre TDD (Test Driven Development) e sua aplicação prática.

## Descrição do Desafio

O desafio proposto neste módulo é explorar o potencial dos testes unitários e entender como eles podem contribuir para a criação de código mais robusto e confiável. Durante o curso, aprendemos sobre:

- A importância dos testes unitários na garantia da qualidade do código.
- Técnicas de TDD (Test Driven Development) para desenvolvimento orientado a testes.
- Estratégias para escrever testes eficazes e abrangentes.
- Utilização de ferramentas como xUnit para escrever e executar testes unitários.

Você é desafiado a criar seu próprio projeto ou reproduzir o que foi feito durante o módulo, aplicando os conceitos e técnicas aprendidas.

## Funcionalidades

Este projeto implementa uma calculadora geométrica com as seguintes funcionalidades:

- Cálculo da área de um círculo.
- Cálculo do perímetro de um círculo.
- Cálculo do diâmetro de um círculo.

## Tecnologias Utilizadas

- C#
- .NET Core
- xUnit

## Como Executar os Testes

Para executar os testes unitários, siga os passos abaixo:

1. Clone o repositório para o seu ambiente local.
2. Abra o projeto CalculadoraGeometrica.Tests em um ambiente de desenvolvimento C#.
3. Execute os testes utilizando a ferramenta de teste do seu ambiente de desenvolvimento ou através do terminal usando o comando `dotnet test`.

## Meta de Cobertura de Testes

A meta de cobertura de testes para este projeto é de 80%. Você pode acompanhar o progresso da cobertura através da ferramenta de cobertura de código integrada ou através da linha de comando.
